Room Quest

Augmeneted Reality Game

Using Unity and the Vuforia Engine, I developed an augmented reality game. The goal of the game is to score as many baskets as you can within a certain time limit. How the game works: once the user has a specific image in the camera's view, a basketball hoop and backboard will appear and the user will be able to shoot the ball in. As the player scores, they are able to keep track of how many shots are made.
